Markdown Here Options: CSS Merge

First of all, don't panic. You're here because Markdown Here's default "Primary Styling CSS" has changed. This is probably to fix a bug or make stuff look better. In you're interested, you can check out the history of the default CSS.

If you haven't ever made changes to your CSS, you should probably just click the right-hand side button that will save the default CSS as your Primary Styling CSS.

If you have customized your CSS, or if you're a naturally inquisitive sort, you can merge bits and pieces from one side into the other, check out the preview, and then choose the side you want to use.

If you don't feel like doing this right now, or if you mess something up, just close this tab. You'll have the choice to do this again the next time you open the Markdown Here Options tab.

If you don't want to change anything, and you don't want to see this again (unless the default CSS changes again), just click the left-hand side button that will keep your current CSS.
